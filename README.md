# TruthCraft

TruthCraft is a lightweight, frontend-only React + TypeScript truth table generator and Boolean algebra learning app.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy on Vercel

- Framework Preset: Vite
- Install Command: `npm install`
- Build Command: `npm run build`
- Output Directory: `dist`
- Node.js Version: 20.x or newer

This project includes a tested `package-lock.json` for reproducible installs. Do not upload `node_modules`.
